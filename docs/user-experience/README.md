# User Experience Documentation

This folder contains user experience design, copy, and interaction documentation for the Deduper project.

## Contents

- **ERRORS_AND_UX_COPY.md** - Error messages and user-facing copy standards
- **ONBOARDING_SPEC.md** - User onboarding experience specifications
- **UX_COPY_STYLE.md** - Style guide for user interface copy
- **UX_PERF_BUDGETS.md** - Performance budgets for user experience
- **SHORTCUTS_AND_BATCH_UX.md** - Keyboard shortcuts and batch operation UX
- **EXTERNAL_MEDIA_HANDLING.md** - Handling of external media types

## Purpose

These documents ensure:
- Consistent user experience across the application
- Clear and helpful error messaging
- Efficient user workflows and interactions
- Performance standards that impact user experience
- Proper handling of various media types
