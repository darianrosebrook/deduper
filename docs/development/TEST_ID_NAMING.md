## Test ID Naming
Author: @darianrosebrook

### Pattern

`<Area>_<Case>_<NNN>`

Areas: Scan, Meta, Hash, Video, Group, Merge, Undo, UI, Bench

Examples:

- Scan_BundlesExcluded_001
- Meta_ExifParse_JPEG_002
- Hash_Stability_pHash_003
- Video_FrameExtract_ShortClip_004
- Group_DistanceThreshold_Boundary_005
- Merge_ExifWrite_GPSFill_006
- Undo_MultiStep_007
- UI_EvidencePanel_Expand_008
- Bench_TimeToFirstGroup_Medium_009
